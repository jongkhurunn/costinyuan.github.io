操作符重载（成员函数）

  • 重载的返回值设置成 complex& 类型而不是 void， 是考虑到连加的情况

操作符重载（非成员函数）

  • 不可以return by reference，因为返回的是一个local object

  • 临时对象 typename ( ) 的用法，生命周期到函数结束

  • << 必须写作非成员函数

    • 连续输出所以返回类型要设置成 ostream&

总结注意

  1. 数据一定放在private
  2. 参数尽可能以reference传
  3. 返回值尽可能用reference传
  4. 类的body中应该加const就要加
  5. 构造函数的initialization list
